# 🌏 アジア / Asia

アジアの収録国一覧と分類ルールです。

## 📁 ディレクトリ構造

```
asia/
├── east_asia/          # 東アジア
├── southeast_asia/     # 東南アジア
├── south_asia/         # 南アジア
├── central_asia/       # 中央アジア
└── west_asia/          # 西アジア
```
