# 🗺️ ロードマップ / Roadmap

このドキュメントでは、World Address YAML プロジェクトおよび **Veyエコシステム** の現在の実装状況と将来の計画を示します。

**Vey**（ヴェイ）は "convey"（配達する）に由来し、メールのような配送体験とクレジットカードのような利便性を目指すエコシステムです。詳細は [Veyエコシステムドキュメント](./docs/vey-ecosystem.md) をご覧ください。

## 📊 現在の状況 (2024年12月)

### ✅ 実装済み / Implemented

| 機能 | 詳細 | バージョン |
|------|------|-----------|
| **データ収録** | 247国・地域のYAML/JSONデータ | v1.0 |
| **libaddressinput 自動取得** | Google libaddressinput からのデータ自動取得・更新（毎日深夜0時JST） | v1.0 |
| **データバリデーション** | YAML構文・必須フィールドの自動検証スクリプト | v1.0 |
| **GitHub Actions 統合** | PR/Push時の自動バリデーション | v1.0 |
| **データ統計機能** | 完成度の可視化、フルスキーマ対応国の一覧 | v1.0 |
| **ライセンス明記** | MIT License | v1.0 |
| **SDK コア開発** | @vey/core パッケージ（ローカル開発中） | v1.0-alpha |
| **完全実装例** | JP、USの完全実装例 | v1.0 |

## 🎯 長期計画 (12-24ヶ月)

### v2.0 - Veyvault MVP（クラウド住所帳）

**目標:** ZKP ベースのプライバシー保護型住所管理 - **Veyvault** アプリケーションのリリース

**Veyvault**（ヴェイブック）は、ユーザーの住所を安全に管理するクラウド住所帳アプリです。

- [ ] AMF (Address Mapping Framework) 実装
- [ ] PID (Place ID) 生成・管理システム
- [ ] エンドツーエンド暗号化
- [ ] Verifiable Credential (VC) 発行
- [ ] 基本的なフレンド管理機能
- [ ] ソーシャルログイン統合（Google、Apple、LINE）
- [ ] Veyvaultモバイルアプリ（iOS/Android）リリース
- [ ] Veyvault Web アプリリリース

### v2.1 - ZKP プロトタイプ

**目標:** ゼロ知識証明による配送検証

- [ ] ZK Circuit 実装
- [ ] 国コード・地域レベルのZK証明
- [ ] ECサイト統合フロー（プロトタイプ）
- [ ] 配送可能性検証（住所非公開）
- [ ] Veyvault APIでのZKP証明生成サポート

### v2.5 - Veypos MVP（POSアプリケーション）

**目標:** グローバルPOSシステムのリリース

**Veypos**（ヴェイポス）は、世界中の税制・通貨・レシート要件に対応したPOSアプリです。

- [ ] 多通貨・多税制対応エンジン
- [ ] レシート発行システム（各国の法的要件対応）
- [ ] Veyvault連携（QR/NFCでの住所登録）
- [ ] 決済処理統合
- [ ] Veyposモバイルアプリ（iPad/タブレット）リリース
- [ ] 店舗管理ダッシュボード

### v3.0 - 配送統合

**目標:** 実用的な配送システム統合

- [ ] 配送業者API統合
- [ ] 高度なアクセス制御
- [ ] 完全な監査ログ
- [ ] ラストワンマイル住所開示
- [ ] VeyvaultからEC連携での配送追跡

### v4.0 - DID/VC 完全連携

**目標:** 分散型ID・検証可能クレデンシャル対応

- [ ] DID標準準拠
- [ ] VC発行・検証システム
- [ ] マルチキャリア対応
- [ ] グローバル相互運用性
- [ ] Veyvault Verifiable Credential発行システム

### v5.0 - モバイルウォレット統合

**目標:** Google Wallet / Apple Wallet 統合

- [ ] Google Wallet 完全統合
- [ ] Apple Wallet 完全統合
- [ ] QR/NFC 配送証明
- [ ] ホテル・金融機関での住所証明
