# Vey エコシステム ユーザージャーニー / User Journeys

このドキュメントは、Veyエコシステムにおける主要なユーザージャーニーを示します。

This document illustrates the primary user journeys in the Vey ecosystem.

---

## 👤 ユーザーペルソナ / User Personas

### 1. 一般消費者 (田中さん) / General Consumer (Tanaka-san)
- **年齢**: 28歳
- **職業**: 会社員
- **ニーズ**: オンラインショッピングを頻繁に利用、住所入力が面倒

### 2. ECサイト運営者 (佐藤さん) / E-commerce Operator (Sato-san)
- **年齢**: 35歳
- **職業**: EC事業者
- **ニーズ**: 配送業務の効率化、国際配送の拡大

### 3. 配送業者 (山田さん) / Delivery Worker (Yamada-san)
- **年齢**: 42歳
- **職業**: 配送ドライバー
- **ニーズ**: 配送ルートの最適化、効率的な配達

### 4. 小売店オーナー (鈴木さん) / Retail Store Owner (Suzuki-san)
- **年齢**: 50歳
- **職業**: 小売店経営
- **ニーズ**: 在庫管理、POS統合、配送手配
