# Veyform Developer Console

The Developer Console provides a visual interface for configuring Veyform for your site without writing code.

## Features

1. **Country Setup** - Select which countries to support
2. **Language Configuration** - Choose default and available languages
3. **Hierarchy Preview** - Visualize address fields for each country
4. **Code Generation** - Auto-generate integration code for React/Vue/HTML
5. **Analytics Dashboard** - View usage statistics

## Access

Console URL: https://console.vey.dev
