## 📮 郵便運用について

南極には以下の特殊ルールがあります：

- 郵便番号制度なし
- 国境なし（南極条約）
- 郵便は各国の本土郵便局を経由
- 実務住所は「宛名＋基地名＋国＋経由地」の形式
